Selecting with jQuery

Title can change

You can use the $ sign to select elements It will feel a lot like selecting using querySelectorAll()

For example :

  • Selecting all a tags:
    • $('a')
  • Selecting all elements for the class "container"
    • $(".container")
  • Selecting all elements with id "special"
    • $("#special")
